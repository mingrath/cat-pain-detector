# Model Card Draft

## Intended Use

Triage-support prototype for identifying visible feline pain indicators from cat face images.

## Not Intended For

- Veterinary diagnosis
- Emergency decision replacement
- Medication guidance
- Use when the cat is not visible, heavily occluded, sleeping, grooming, or in a non-frontal pose without uncertainty warning

## Validation Status

Pending FGS-labeled validation data. CatFLW validates landmark/evidence pipeline only.
